# 金融监管知识图谱系统

基于人工智能的金融监管知识图谱构建与管理平台。

## 项目概述

本系统提供金融监管领域文档的知识图谱构建、可视化和智能问答功能，帮助用户高效管理和分析监管法规、风险指标、金融机构等实体之间的关系。

## 功能模块

| 模块 | 说明 | 入口文件 |
|------|------|----------|
| 首页仪表盘 | 系统概览和快捷入口 | `index.html` |
| 文档管理 | 文档上传、分类、查看、构建 | `docs.html` |
| 图谱管理 | 节点、关系、三元组管理 | `kg-manage.html` |
| 图谱可视化 | 交互式图谱可视化 | `kg-vis.html` |
| 智能问答 | 基于知识图谱的智能问答 | `qa.html` |
| 用户管理 | 用户和权限管理 | `users.html` |

## 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 无需安装，直接打开 HTML 文件

### 启动方式

#### 方式一：直接打开
```bash
# 进入项目目录
cd kg-system

# 直接打开 index.html
open index.html
# 或双击 index.html 文件
```

#### 方式二：本地服务器
```bash
# 使用 Python
cd kg-system
python -m http.server 8080
# 访问 http://localhost:8080

# 或使用 Node.js
npx serve .
```

## 数据说明

### 预置数据
系统预置了金融监管领域的示例数据：

- **节点**：89 个实体
- **边**：78 条关系
- **节点类型**：12 种
- **关系类型**：14 种

### 数据领域
- 监管机构（银保监会、证监会等）
- 法规文件（资本管理办法、流动性管理办法等）
- 金融机构（商业银行、证券公司等）
- 监管指标（资本充足率、流动性覆盖率等）
- 风险类别（信用风险、流动性风险等）
