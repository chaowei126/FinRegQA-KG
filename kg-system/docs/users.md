# 图谱可视化

## 功能概述

图谱可视化模块提供知识图谱的交互式可视化展示，支持多种布局方式和丰富的交互功能。

## 布局模式

### 1. 力导向布局（默认）
- 模拟物理力学，节点自动排列
- 关联节点相互吸引
- 无关联节点相互排斥
- 适合展示复杂关系网络

### 2. 圆形布局
- 所有节点排列在一个大圆上
- 节点均匀分布
- 适合展示全局概览

### 3. 辐射布局
- 高度数节点位于中心
- 低度数节点位于外圈
- 按节点重要性分层
- 适合突出核心实体

## 显示控制

### 数量限制
- 显示50条
- 显示100条（默认）
- 显示200条
- 显示全部

### 关系过滤
顶部关系类型过滤栏，可勾选/取消勾选各类关系：
- 监管、颁布、遵循、制定、评估
- 包含、申请、产生、面临、影响
- 需满足、计算方式、指标、关联

## 交互功能

### 节点操作
| 操作 | 说明 |
|------|------|
| 点击节点 | 显示节点详情面板 |
| 拖拽节点 | 拖拽后节点固定位置 |
| 释放节点 | 节点恢复自由浮动 |
| 双击节点 | 重置节点位置 |

### 画布操作
| 操作 | 说明 |
|------|------|
| 拖拽空白处 | 平移画布 |
| 滚轮 | 缩放画布 |
| 暂停按钮 | 暂停/继续力导向模拟 |

### 节点详情面板
点击节点后显示：
- 节点名称
- 节点类型（带颜色标识）
- 关联节点数量
- 关联关系列表（前5条）

### 显示模式
- **全图**：显示完整知识图谱
- **按文档**：按来源文档分组
- **按类型**：按节点类型着色
- **按关系**：按关系类型着色

## 搜索功能

### 搜索节点
1. 在搜索框输入关键词
2. 选择搜索深度（1级/2级/3级关系）
3. 点击「查找」或按 Enter

### 搜索结果
- 高亮匹配的节点
- 显示匹配的节点及其指定深度的关联
- 显示匹配数量统计

## 快捷操作

| 操作 | 说明 |
|------|------|
| ↺ 刷新图谱 | 重新加载并渲染图谱 |
| ⟲ 重置布局 | 清除搜索和筛选，恢复默认视图 |

## 技术说明

### 渲染引擎
- 纯 SVG 渲染，无需外部依赖
- 自定义力导向模拟算法
- 支持大规模节点（500+）

### 性能优化
- 圆形边界约束，防止节点溢出
- 节点压缩算法，避免重叠
- 按需渲染，支持数据量限制

### 节点大小
节点半径根据度数（关联数量）计算：
```
半径 = 10 + (度数 / 最大度数) × 15
```
度数越高的节点，显示越大。
