# 文档管理

## 功能概述

文档管理模块提供金融监管文档的上传、分类、查看和知识图谱构建功能。

## 主要功能

### 1. 目录管理
- 创建新目录
- 重命名目录
- 删除目录（会同时删除目录下所有文档）
- 目录展开/收起

### 2. 文档上传
- 支持格式：PDF、Word、Excel、TXT
- 文件大小限制：单文件不超过 50MB
- 可选择上传到指定目录

### 3. 文档查看
提供三种查看模式：

#### 原文档
- PDF 文档：左侧缩略图 + 右侧内容预览
- Excel 文档：提示不支持在线预览，可下载查看
- Word 文档：文本内容预览

#### 提取内容
- 显示文档摘要
- 关键条款展示
- 监管指标表格（如有）

#### 知识图谱
- 显示该文档构建的知识图谱
- 点击"全屏查看"可跳转到图谱可视化页面

### 4. 文档筛选
顶部统计栏提供多种筛选方式：
- 按文件类型：PDF / Word / Excel
- 按构建状态：已构建 / 未构建
- 搜索：支持按标题、摘要、文件名搜索

### 5. 知识图谱构建
点击文档卡片中的"构建知识图谱"按钮，可选择：
- **构建当前文档**：仅构建当前选中的文档
- **按目录构建**：构建当前目录下所有文档
- **全量构建**：构建所有未构建的文档

## 操作入口

| 操作 | 入口 | 说明 |
|------|------|------|
| 上传文档 | 侧边栏「↑ 上传文档」按钮 | 支持拖拽上传 |
| 新建目录 | 侧边栏「⊞ 新建目录」按钮 | 输入目录名称 |
| 查看文档 | 点击左侧文件列表 | 显示文档详情 |
| 构建图谱 | 文档查看页「⚡ 构建知识图谱」按钮 | 启动AI构建 |

## 状态说明

| 状态 | 图标 | 说明 |
|------|------|------|
| 已构建 | ● (绿色) | 知识图谱已生成 |
| 未构建 | ○ (灰色) | 尚未生成知识图谱 |
