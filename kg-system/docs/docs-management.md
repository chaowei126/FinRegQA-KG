# 图谱管理

## 功能概述

图谱管理模块提供知识图谱的全局视图、节点管理、关系管理和三元组编辑功能。

## 主要功能

### 1. 图谱概览
提供知识图谱的统计信息和结构分布：
- **节点总数**：知识图谱中的实体数量
- **关系总数**：实体之间的关联数量
- **节点类型分布**：各类实体的数量统计
- **关系类型分布**：各类关系的数量统计

### 2. 节点管理
管理知识图谱中的所有节点（实体）：

#### 功能
- 查看所有节点列表
- 按名称搜索节点
- 按类型筛选节点
- 查看节点的关联关系数
- 编辑节点信息
- 删除节点

#### 节点属性
| 属性 | 说明 |
|------|------|
| 节点名称 | 实体的标识名称 |
| 节点类型 | 实体的类别（如：监管机构、金融机构、法规文件等） |
| 关联关系数 | 该节点与其他节点的连接数量 |
| 来源文档 | 该节点从哪个文档提取 |

### 3. 关系管理
管理知识图谱中的所有关系（边）：

#### 功能
- 查看所有关系类型统计
- 查看各类型关系的示例
- 按类型筛选关系

#### 预定义关系类型
- 监管、颁布、遵循、制定、评估
- 包含、申请、产生、面临、影响
- 需满足、计算方式、指标、关联

### 4. 三元组管理
以「头实体 - 关系 - 尾实体」形式管理知识图谱：

#### 功能
- 查看所有三元组
- 搜索：按头实体、关系、尾实体搜索
- 添加新三元组
- 编辑现有三元组
- 删除三元组

#### 三元组表单
- 头实体名称
- 头实体类型
- 关系类型
- 尾实体名称
- 尾实体类型

### 5. 类型管理
管理系统中的节点类型和关系类型：

#### 节点类型
| 类型 | 颜色 | 说明 |
|------|------|------|
| 监管机构 | 蓝色 | 如：银保监会、证监会 |
| 法规文件 | 红色 | 如：资本管理办法 |
| 金融机构 | 绿色 | 如：商业银行、证券公司 |
| 监管指标 | 橙色 | 如：资本充足率 |
| 风险类别 | 紫色 | 如：信用风险、流动性风险 |
| 处罚事项 | 粉色 | 如：行政处罚 |
| ... | ... | ... |

## 数据结构

### 节点类型定义
```javascript
{
  '监管机构': '#1677ff',
  '法规文件': '#f5222d',
  '金融机构': '#52c41a',
  '监管指标': '#fa8c16',
  '风险类别': '#722ed1',
  '处罚事项': '#eb2f96',
  // ...
}
```

### 节点数据结构
```javascript
{
  id: '节点名称',
  type: '节点类型'
}
```

### 边数据结构
```javascript
{
  s: '头实体',
  t: '尾实体',
  r: '关系类型'
}
```

## 快捷操作

| 操作 | 说明 |
|------|------|
| + 新增节点 | 添加新的实体到图谱 |
| + 新增关系 | 添加新的三元组 |
| 📥 批量导入 | 从文件导入数据 |
| 📤 批量导出 | 导出图谱数据 |
| 📊 打开可视化 | 跳转到图谱可视化页面 |
