# 用户管理

## 功能概述

用户管理模块提供系统用户的管理功能，包括用户信息维护、角色权限配置等。

## 用户列表

### 显示信息
- **头像**：用户名首字母，颜色根据角色确定
- **用户名**：用户的显示名称
- **角色**：用户在系统中的权限角色
- **邮箱**：用户的登录邮箱
- **状态**：当前账户状态
- **最后登录**：最近一次登录时间

### 状态说明
| 状态 | 图标 | 说明 |
|------|------|------|
| 活跃 | ● 绿色 | 账户正常使用 |
| 停用 | ○ 灰色 | 账户已停用，无法登录 |

## 角色权限

### 角色说明

| 角色 | 颜色 | 说明 |
|------|------|------|
| 管理员 | 🔵 蓝色 | 完整系统访问权限，包括用户管理和权限配置 |
| 分析师 | 🟢 绿色 | 文档管理、图谱构建、问答、数据导出 |
| 合规专员 | 🟠 橙色 | 文档查看、图谱可视化、问答、报告生成 |
| 访客 | ⚪ 灰色 | 只读访问，仅支持问答功能 |

### 权限矩阵

| 功能 | 管理员 | 分析师 | 合规专员 | 访客 |
|------|--------|--------|----------|------|
| 首页仪表盘 | ✓ | ✓ | ✓ | ✓ |
| 文档管理 | ✓ | ✓ | ✓ | - |
| 图谱管理 | ✓ | ✓ | ✓ | - |
| 图谱可视化 | ✓ | ✓ | ✓ | ✓ |
| 智能问答 | ✓ | ✓ | ✓ | ✓ |
| 用户管理 | ✓ | - | - | - |
| 数据导出 | ✓ | ✓ | - | - |

## 用户操作

### 新增用户
1. 点击「+ 新增用户」按钮
2. 填写用户信息：
   - 用户名（必填）
   - 邮箱（必填）
   - 角色（默认：访客）
3. 点击「保存」完成创建

### 编辑用户
点击用户列表中的「编辑」链接，修改用户信息。

### 启用/停用
点击用户列表中的「停用/启用」链接，切换用户状态：
- 停用：用户无法登录系统
- 启用：恢复用户登录权限

### 删除用户
点击用户列表中的「删除」链接，确认后删除用户：
- 删除操作不可恢复
- 用户的问答记录将被保留

## 用户统计

页面顶部显示用户统计信息：
- **用户总数**：系统中所有用户数量
- **活跃用户**：状态为「活跃」的用户数量
- **停用用户**：状态为「停用」的用户数量

## 安全建议

### 密码管理
- 建议用户定期更换密码
- 密码强度应符合安全要求
- 不要在多个系统使用相同密码

### 权限控制
- 根据工作需要分配最小权限
- 定期审查用户权限配置
- 离职/调岗时应及时调整权限

### 审计日志
系统会记录以下操作：
- 用户登录/登出
- 权限变更
- 敏感操作

## 数据结构

```javascript
{
  id: 1,                    // 用户ID
  name: '管理员',           // 用户名
  role: 'admin',            // 角色
  email: 'admin@example.com', // 邮箱
  status: 'active',         // 状态
  lastLogin: '2026-05-13 17:00' // 最后登录
}
```
